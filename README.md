## 🚀 Features

| Category | Description |
|----------|-------------|
| 🔍 **Reconnaissance** | Subdomain discovery, directory brute‑force, port scanning, JS secret extractor, HTTP header analysis, CORS misconfiguration, technology fingerprinting |
| 💣 **Exploitation** | SQL injection, XSS, LFI, RFI, CSRF, SSRF, XXE, deserialization – 100+ ready‑made payloads |
| 🛡️ **Defensive Checks** | TLS/SSL analysis, cookie flags, HSTS, CSP, mixed content, robots.txt/sitemap analysis |
| 🌐 **Network** | Live packet sniffing, PCAP export, DNS lookup, ping, traceroute, port scanning |
| 💻 **System** | CPU, memory, process list, file system access, registry read/write (Windows), shell command execution |
| 🔥 **Threat Intelligence** | IP tracking, Shodan/AlienVault/VirusTotal simulation, automated reports |
| 🌑 **Darkweb** | .onion detector, Tor proxy support (configurable) |
| ⛓️ **Blockchain** | Bitcoin/Ethereum wallet scanner, transaction analysis (conceptual) |
| 🐉 **Visual** | Cyberpunk UI, glassmorphism, live dragon animation (mouse‑following + tail) |
| ⚡ **Performance** | Parallel tool execution, minimal memory footprint, real‑time feed |
